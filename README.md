# 👨‍💻 Currículo Web Interativo

Um currículo digital responsivo e animado, construído para apresentar meu portfólio, habilidades e informações de contato de forma dinâmica e profissional.

Este projeto foi desenvolvido a partir de um template base, onde implementei melhorias estruturais, personalização de UI e integrei APIs externas para facilitar o contato direto.

## 🚀 Funcionalidades e Integrações

- **📱 Integração com WhatsApp API:** Botão de contato direto que abre uma conversa no WhatsApp instantaneamente.
- **🗺️ Integração com API de Mapas:** Endereço interativo que redireciona o usuário para a localização exata no mapa.
- **✨ Animações on Scroll:** Elementos da página surgem de forma suave conforme o usuário desce a tela, melhorando a UX.
- **📱 Design Responsivo (Mobile First):** Layout totalmente adaptável para smartphones, tablets e desktops utilizando Bootstrap.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3 & Bootstrap:** Estilização avançada, sistema de grid e componentes responsivos.
- **JavaScript (ES6+):** Lógica para animações de rolagem e manipulação do DOM.
- **APIs Externas:** WhatsApp API e integração de Mapas.
- **Ferramentas:** Git, GitHub, VS Code.
